Model Building

Use machine learning methods for property prediction.

How do machine learning models work?

At the manufacturing plant, you have now been tasked with determining the toxicity of certain fuels. The toxicity of certain substances depends on their lipophilicity. This is a measure of how well they can dissolve in fats or other nonpolar substances, which is associated with their ability to cross the cell membrane and potentially kill cells.

You'll be using a machine learning model to predict lipophilicity using a dataset of known values. At its simplest, this is a form of high-level regression.

Use a Line Graph

We will be using data about alkanes from https://www.engineeringtoolbox.com/hydrocarbon-boiling-melting-flash-autoignition-point-density-gravity-molweight-d_1966.html

As the manager, you have access to various properties about different types of alkanes:

    # Carbon Atoms # Hydrogen Atoms Molecular Weight Density

However, you notice that you are missing boiling point data about certain alkanes. Your task is to use the rest of the data to estimate the missing information.

Regression

Using a line graph for interpolation risks misjudging the data. Instead, we can use a computer to generate a curve of best fit. This is known as regression and will allow you to fill in missing values that best fit all of the data.

You may have noticed that multiple quantities (known as features) serve well as predictors. Now we can analyze these relationships mathematically by forming a quadratic regression equation.

boiling point = -0.9859(# carbon atoms)^2 + 45.1279(# carbon atoms) + -176.8114

Coefficient of determination (R^2): 0.995

Note: The coefficient of determination represents how well the curve fits the data. A value of 1 is perfect.
